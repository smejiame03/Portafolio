Proyectos desarrollados


A continuación se observan los proyectos que he desarrollado.


QR Assistance

Proyecto académico desarrollado para la toma de asistencia estudiantil por medio de un código QR.


Proyecto desarrollado con HTML y CSS, funciones con JavaScript, base de datos MySQL y conexión a la DB con PHP.


NFT. Punto de venta

Proyecto académico desarrollado para la gestión administrativa de un punto de venta.


Proyecto desarrollado con HTML, CSS y BOOTSTRAP, funciones con JavaScript, base de datos MySQL y conexión a la DB con PHP.


Servicio de Transporte

Proyecto académico desarrollado para ofrecer y coordinar el servicio de transporte como conductor o pasajero.

Proyecto desarrollado con HTML y CSS, funciones con JavaScript, base de datos MySQL y conexión a la DB con PHP.


CIVILIZATE

Proyecto académico desarrollado en conjunto con estudiantes de ingeniería civil para determinar las condiciones de las viviendas de Medellín y la escala de gravedad de las mismas para diagnosticar su grado de vulnerabilidad.

Proyecto desarrollado con HTML y CSS, funciones con JavaScript, base de datos MySQL y conexión a la DB con PHP.

Chat Empresarial

Proyecto académico desarrollado para facilitar la comunicación entre los empleados de una empresa por medio de un chat, al cual ingresan ya sea por medio de un login o con reconocimiento facial.

Proyecto desarrollado con PYTHON, y librerías como TKINTER, OpenCV, Matplotlib, Numpy, Socket programming, Time y MTCNN para el reconocimiento facial.


SAMI

Sistema experto para el control y tratamiento de trastornos emocionales (ansiedad, depresión y estrés) en estudiantes universitarios.
